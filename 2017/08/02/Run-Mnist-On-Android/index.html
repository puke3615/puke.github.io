封装输出数据解析逻辑

在手写数字识别模型中的输出是一个size为10的列表，列表元素的索引值对应输出的结果，列表元素对应输出的概率，例如输出是[0.2, 0.7, 0.01……]，即表示有0.2的概率是0，0.7的概率是1，0.01的概率是2……

因此我们需要在输出中对数据按照概率进行降序排列，以便让结果一目了然。

这时我们就能通过MnistData类的top方法得到概率最大的几个结果分别是什么。
